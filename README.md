# 🤖 Atlas

Atlas — локальный AI-ассистент с поддержкой инструментов, работающий на Ollama (в будущем на gemini и других AI).

## Возможности

- 💬 Telegram Bot
- 🧠 Локальная LLM (Ollama)
- 🛠 Система инструментов
- 📂 Работа с файлами
- 📝 Планировщик действий
- 🔌 Расширяемая архитектура

## Стек

- Python
- Ollama
- python-telegram-bot
- Git

## Запуск

```bash
pip install -r requirements.txt
python main.py
```
